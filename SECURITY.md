# 🔒 安全策略 (Security Policy)

## 报告安全问题

如果您发现本项目存在安全漏洞或问题，请：

### ✅ 正确做法
1. **发送邮件至**: rao5201@126.com
2. **包含详细信息**: 
   - 问题描述
   - 复现步骤
   - 影响范围
   - 建议修复方案（如有）
3. **等待确认**: 我们会在 48 小时内回复
4. **保密**: 在修复前不要公开披露

### ❌ 错误做法
- 不要在公开 Issues 中报告安全问题
- 不要在论坛或社交媒体讨论
- 不要利用漏洞进行测试
