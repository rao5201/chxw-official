## 🔐 Supabase 数据库安全

### 当前配置
```javascript
✅ 使用 Publishable Key（公开密钥）
✅ 仅具有 SELECT 权限
✅ 无法修改或删除数据
✅ 适合前端直接使用
```

### 建议配置（RLS 行级安全）

在 Supabase SQL Editor 执行：

```sql
-- 启用新闻表 RLS
ALTER TABLE news ENABLE ROW LEVEL SECURITY;
CREATE POLICY "公开读取" ON news FOR SELECT
  USING (is_published = true);

-- 启用产品表 RLS
ALTER TABLE products ENABLE ROW LEVEL SECURITY;
CREATE POLICY "公开读取" ON products FOR SELECT
  USING (is_published = true);

-- 启用工具表 RLS
ALTER TABLE tools ENABLE ROW LEVEL SECURITY;
CREATE POLICY "公开读取" ON tools FOR SELECT
  USING (is_published = true);
```

**效果**:
- 🔒 只能读取已发布内容
- 🔒 无法访问未发布数据
- 🔒 防止数据篡改
