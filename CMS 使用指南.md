## 📋 系统概述

茶海虾王内容管理系统 (CMS) 是一款基于 Supabase 数据库的智能化管理工具，提供产品上传、新闻发布、工具管理和 AI 自动处理功能。

### 核心功能
- 🛒 **产品管理** - 上传、分类、定价、促销标签
- 📰 **新闻管理** - 发布、描述、链接、日期管理
- 🔧 **工具管理** - 添加、描述、下载链接
- 🤖 **AI 自动处理** - 智能生成、优化、分类、SEO

## 📖 功能详解

### 🛒 产品管理

#### 上传产品步骤
1. 点击"产品管理"选项卡
2. 填写产品信息：
   - **产品名称** *（必填）
   - **价格 (¥)** *（必填）
   - **产品分类** *（必选）
     - 🔒 数字安全
     - 🥬 农副产品
     - 🚗 新能源汽车
     - 🏭 生产资料
     - 📦 其他产品
   - **产品图标**（可选，建议用 Emoji）
   - **促销标签**（可选，如：新品、热销、爆款）
   - **购买链接**（可选）
   - **立即发布**（默认勾选）

3. 点击"📤 上传产品"按钮
4. 等待成功提示 ✅

#### 示例数据
```
产品名称：有机绿茶
价格：198.00
分类：🥬 农副产品
图标：🍵
标签：新品
购买链接：https://example.com/product/123
```

### 🤖 AI 自动处理

#### AI 功能列表

##### 1. 📰 自动生成新闻摘要
**用途**: 根据标题自动生成完整的新闻摘要

**输入示例**:
```

```

**AI 输出**:
```

标题：茶海虾王官方网站 v1.0.2 发布

关键词：新闻，热点，资讯

SEO 优化：茶海虾王官方网站 v1.0.2 发布 | 茶海虾王资讯中心
```

**用途**: 优化产品描述，提炼卖点

```
有机绿茶 2026 新茶
```

```
【AI 优化产品描述】

原始：有机绿茶 2026 新茶

优化后：有机绿茶 2026 新茶 - 高品质产品，值得信赖！采用先进工艺，性能卓越，性价比高，是您理想的选择。

卖点提炼：
• 品质保证
• 性能卓越
• 性价比高
• 售后完善
```

##### 3. 🏷️ 自动分类产品
**用途**: 根据产品名称自动推荐分类

**输入示例**:
```
特斯拉 Model 3 充电桩
```

**AI 输出**:
```
【AI 自动分类】

产品：特斯拉 Model 3 充电桩

推荐分类：auto (新能源汽车)

分类依据：包含关键词"汽车"
```

**支持的关键词**:
- 茶、食品 → 🥬 农副产品
- 软件、安全 → 🔒 数字安全
- 汽车 → 🚗 新能源汽车
- 设备 → 🏭 生产资料

**用途**: 优化标题，提升搜索引擎排名

**输入示例**:
```

```

**AI 输出**:
```
【AI SEO 优化】

原标题：网络安全防护软件

优化后：网络安全防护软件 - 2026 最新 | 茶海虾王官方推荐

关键词建议：网络安全防护软件，十大品牌，评测，价格，购买指南

Meta 描述：了解网络安全防护软件的详细信息，包括价格、性能、用户评价等。茶海虾王为您提供专业的产品推荐和购买指南。
```

1. 点击"AI 自动处理"选项卡

3. 输入原始内容

## 🔐 安全说明

### 数据库权限
- ✅ 使用 Publishable Key（公开密钥）
- ✅ 仅具有数据读取和插入权限
- ✅ 无法删除或修改现有数据
- ✅ 适合前端直接使用

### 数据安全
- ⚠️ 不要在前端暴露 Service Role Key
- ⚠️ 定期备份数据库
- ⚠️ 启用 RLS 行级安全策略
- ⚠️ 限制未授权访问

## 🚨 常见问题

### Q1: 上传失败怎么办？
**A**: 检查以下几点：
1. 网络连接是否正常
2. Supabase 数据库是否可访问
3. 必填字段是否完整
4. 查看错误提示信息

### Q2: AI 处理不准确？
**A**: AI 功能基于规则匹配，建议：
1. 输入更详细的描述
2. 手动调整分类
3. 结合人工审核

### Q3: 统计数据不更新？
**A**: 刷新页面或点击统计卡片重新加载

### Q4: 如何修改已发布内容？
**A**: 当前版本仅支持新增，修改需要：
1. 在 Supabase Dashboard 手动编辑
2. 或开发编辑功能（待扩展）
