## 🤖 AI 功能详解

### 1. 📰 自动生成新闻摘要
**输入**: 新闻标题  
**输出**: 完整新闻摘要 + 关键词 + SEO 优化

**示例**:
```
输入：茶海虾王 v1.0.3 发布
输出：【AI 生成新闻摘要】
      标题：茶海虾王 v1.0.3 发布
      摘要：这是一条重要新闻...
      关键词：新闻，热点，资讯
      SEO：茶海虾王 v1.0.3 发布 | 茶海虾王资讯中心
```

### 2. 🛒 优化产品描述
**输入**: 产品基本信息  
**输出**: 优化描述 + 卖点提炼

**示例**:
```
输入：有机绿茶 2026 新茶
输出：【AI 优化产品描述】
      优化后：高品质产品，值得信赖...
      卖点：品质保证、性能卓越、性价比高
```

### 3. 🏷️ 自动分类产品
**输入**: 产品名称  
**输出**: 推荐分类 + 分类依据

**支持关键词**:
- 茶、食品 → 🥬 农副产品
- 软件、安全 → 🔒 数字安全
- 汽车 → 🚗 新能源汽车
- 设备 → 🏭 生产资料

### 4. 🔍 SEO 优化标题
**输入**: 原始标题  
**输出**: 优化标题 + 关键词建议 + Meta 描述

**示例**:
```
输入：网络安全防护软件
输出：优化后：网络安全防护软件 - 2026 最新 | 茶海虾王官方推荐
      关键词：十大品牌，评测，价格，购买指南
```

## 📝 使用教程

### 快速开始
1. **打开 CMS 系统**
   - 本地：双击 `admin-cms.html`
   - 在线：访问 GitHub Pages 部署后的链接

2. **查看统计数据**
   - 首页显示新闻/产品/工具总数
   - AI 处理次数统计

3. **选择功能模块**
   - 点击选项卡切换（产品/新闻/工具/AI）

4. **填写表单并提交**
   - 必填字段标有 *
   - 实时验证和反馈

### 产品上传示例
```
1. 点击"产品管理"选项卡
2. 填写信息：
   - 产品名称：有机绿茶
   - 价格：198.00
   - 分类：🥬 农副产品
   - 图标：🍵
   - 标签：新品
   - 购买链接：https://...
3. 点击"📤 上传产品"
4. 等待成功提示 ✅
```

### 新闻发布示例
```
1. 点击"新闻管理"选项卡
2. 填写信息：
   - 标题：茶海虾王 v1.0.3 发布
   - 描述：新增 CMS 管理系统
   - 链接：https://github.com/...
   - 日期：2026-04-10
3. 点击"📤 发布新闻"
4. 等待成功提示 ✅
```

### AI 处理示例
```
1. 点击"AI 自动处理"选项卡
2. 选择功能：自动生成新闻摘要
3. 输入：茶海虾王 v1.0.3 发布
4. 点击"🤖 开始 AI 处理"
5. 查看并复制结果
```
