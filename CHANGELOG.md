#### 表结构创建
- **news 表** - 新闻管理

## 版本说明

### 版本号规则
- **主版本号 (Major)**：不兼容的 API 修改
- **次版本号 (Minor)**：向下兼容的功能性新增
- **修订号 (Patch)**：向下兼容的问题修正

### 发布周期
- 主版本：根据重大功能发布
- 次版本：每月发布
- 修订版：按需发布
